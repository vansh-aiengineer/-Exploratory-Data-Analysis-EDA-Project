# Exploratory Data Analysis (EDA) on Movie Database Dataset

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a movie database dataset to uncover patterns, trends, and relationships among different movie attributes.

## Objectives

* Understand the structure of the dataset.
* Perform statistical analysis on movie-related features.
* Identify missing values and data quality issues.
* Visualize distributions of numerical variables.
* Analyze correlations between features.
* Extract meaningful insights from the dataset.

## Dataset

Dataset Used: Movie Database Dataset (`mymoviedb (1).csv`)

## Tools and Libraries

* Python
* Pandas
* Matplotlib
* Seaborn

## Analysis Performed

1. Data Loading and Inspection
2. Statistical Summary
3. Missing Value Analysis
4. Correlation Analysis
5. Distribution Analysis using Histograms
6. Data Visualization using Heatmaps

## Key Findings

* Analyzed numerical features and their distributions.
* Identified relationships between movie-related variables.
* Visualized correlations using a heatmap.
* Explored overall trends present in the dataset.
